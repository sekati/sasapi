destroy

public function destroy():Void

Destroy object elements and events for proper garbage collection. This is a generic destroy method to insure that, at a minimum, the enterFrame and clip itself are removed when destroy is called. This behavior can be overwritten by CoreClip's subclasses. If you wish to use CoreClip's destroy in addition to your subclass destroy you may do so via: 

 
 	super.destroy();
