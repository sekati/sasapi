Class com.sekati.events.Multicaster

  • com.sekati.events.Multicaster

Implemented Interfaces

IMulticastable

Description

The Multicaster Singleton is the middle-ground between Broadcaster which broadcasts to all listeners for all events and Dispatcher which dispatches Events to listeners based on the events they have subscribed to. In other words Multicaster is a Broadcaster with event-subscribed granularity not tied to the Event object or the mx.events.EventDispatcher and is more optimized than Broadcaster.
