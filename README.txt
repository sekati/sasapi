THE SEKATI ACTIONSCRIPT API (SASAPI) FRAMEWORK - http://sasapi.googlecode.com - 2007

------------------------------------------------------------------------------------

I. Introduction

	SASAPI is a multipurpose Actionscript 2.0 framework designed to bootstrap the 
	development process for flash (>=8) sites and applications. It is the 
	predecessor to the SEKATI (AS3) API and boasts a flexible structure, large 
	collection of useful classes, utilities, test files and application skeletons.

	Project Page:	http://sasapi.googlecode.com/
	SVN Repository:	http://sasapi.googlecode.com/svn/trunk/


II. Documentation
	
	Specific attention has been paid to writing thorough and concise Documentation 
	for SASAPI and generating useful Unit Tests to allow the framework to be easily 
	tapped and referenced as needed.
	
	Documentation:	http://docs.sekati.com/sasapi/
	Unit Tests:		http://api.sekati.com/sasapi/tests/
